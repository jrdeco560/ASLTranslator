Translator Data

We have created databases which hold American Sign Language Images. We have integrated Machine Learning to help identify what letter or word a person is trying to communicate by the images in the database. The software takes the images and displays the letters and/or words on the screen. The English to ASL translation is fairly similar. A person can type words on the software and it will translate and display it in ASL. The current translation accuracy is 92.2%. WE have integrated HTML, CSS, JavaSCript, Swift, Python, TensorFlow, Firebase, and CoreML.
